# 🏋️ FitQuest

A modern fitness tracking web application built using Next.js and Tailwind CSS. FitQuest helps users manage workouts, track progress, and stay consistent with their fitness goals.

---

## 🚀 Features

- 🧑‍💻 User Authentication (Login/Signup)
- 📊 Dashboard to track fitness progress
- 🏃 Workout tracking and management
- 📱 Fully responsive design
- ⚡ Fast and optimized performance with Next.js
